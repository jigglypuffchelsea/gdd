# 戰略計畫軌道 — 完整設計規格
**版本：** v1.0
**日期：** 2026-04-03
**狀態：** 待紙面原型驗證

---

## 系統概覽

戰略計畫軌道讓玩家能主動發起跨季度的長期策略目標，取代被動等待卡牌的遊戲感受。

**設計哲學：**
- 計畫是「長線投資 vs 短線靈活」的取捨，不是「更好的投資選項」
- 每個計畫都有隱藏的延遲後果（伏筆種子），完成不是純獎勵
- 玩家在不同遊戲狀態下面對不同的可用計畫——沒有「最佳計畫」

## 核心規則

### 基本參數

| 參數 | 值 |
|------|---|
| 可用時間 | Y2Q1 起（最低年份依計畫而定） |
| 同時上限 | 1 個（挑戰模式可開放至 2 個） |
| 統一工期 | 4 季 |
| 維護費 | +$15 萬/季（加入固定支出） |
| 加碼選項 | 計畫進行至第 2 季里程碑時，可花 1 格 + $50 萬強化完成獎勵 |

### 里程碑機制

**里程碑 = 第 2 季末和第 4 季末（完成）的命脈門檻檢查。**
在八步結算的 Step 5（正常營運）執行：

```
第 2 季里程碑：
  → 相關命脈 ≥ 維護門檻：計畫繼續（可選加碼）
  → 相關命脈 < 維護門檻：計畫停滯

第 4 季里程碑（完成）：
  → 相關命脈 ≥ 維護門檻：計畫完成，交付獎勵，埋伏筆種子
  → 相關命脈 < 維護門檻：計畫失敗（懲罰減半，等同放棄的 50%）
```

### 停滯與救援

- 停滯中：不進展，但每季仍燒 $15 萬維護費
- 救援條件：1 個投資格 + $30 萬 + 相關命脈已回到維護門檻以上
- 連續停滯 2 季 → 自動放棄（保護玩家不無限流血）

### 主動放棄

任何時候可主動放棄，後果：
- 組織心跳 **-5**（團隊被通知中途喊停）
- 已投所有現金沉沒，無退款
- **冷卻 2 季**：期間無法啟動新計畫

### 投資格的機會成本

啟動計畫消耗 1 個投資格。加碼/救援同樣消耗 1 個投資格。
其他季度（進行中）計畫不佔投資格，仍可自由使用 2 格做其他投資。

## 可用性演算法

每季結算前，系統計算哪些計畫可顯示在投資選單：

```
篩選邏輯：
  1. 年份門檻：過濾 Y2+ 或 Y3+ 的要求
  2. 啟動門檻：過濾「需相關命脈 ≥ 啟動門檻」
  3. 排除：已完成的計畫（本局不可重複）
  4. 排除：已有 1 個活躍計畫
  5. 排除：放棄冷卻期中（剩餘 N 季 > 0）

排序邏輯（取前 2 個顯示）：
  優先顯示：對應玩家「最弱命脈」的計畫
  次序：如並列，顯示預付較低的計畫
```

**設計意圖：** 演算法讓玩家看到「針對弱項」的計畫，不是「錦上添花」的計畫。在根系低落時看到新市場計畫，在心跳低落時看到技術壁壘——這防止了收斂到單一最優策略。

## 五個計畫完整設計卡

---

### P1：新市場開拓（Market Expansion）

**核心商業邏輯：** 擴張新市場需要客戶基礎支撐。若組織能力跟不上，新客群反而成為服務崩潰的引爆點。

| 屬性 | 值 |
|------|---|
| **可用年份** | Y2+ |
| **啟動門檻** | 客戶根系 > 45 |
| **維護門檻** | 客戶根系 > 40（里程碑時檢查） |
| **預付成本** | $80 萬 |
| **維護費** | +$15 萬/季（固定支出增加） |
| **工期** | 4 季 |

**完成獎勵（未加碼）：**
- 客戶根系 **+12**
- 季收入永久 **+$20 萬**

**完成獎勵（加碼，$50 萬 + 1 格在 Q2 里程碑投入）：**
- 客戶根系 **+18**
- 季收入永久 **+$30 萬**

**伏筆種子（完成時埋入，+3 季觸發）：**
```
觸發延遲：+3 季
落點類型：條件分支
效果：
  → 若觸發時組織心跳 < 55：
    客戶根系 -8（「新市場客群帶來的期望超出團隊服務能力，老客戶抱怨外溢」）
  → 若組織心跳 ≥ 55：
    無效果（「組織能力追上了，新市場整合成功」）
歸因標籤：新市場開拓後果
溢出處理：×0.7 末季清算
```

**現金流分析：**
```
預付 $80 + 維護 4×$15 = $140 萬總現金投入
完成收益：根系 +12，季收入 +$20 萬
長線收益：每季多 $20 萬，4 季回本，之後淨賺
```

**決策互動（Card #008 跨界競爭者進入）：**
若活躍計畫中，決策牌 #008 的 B 選項（深耕壁壘）效果 +5 根系額外加成（市場開拓已在進行，防禦更有力）。

### P2：技術壁壘（Technology Moat）

**核心商業邏輯：** 技術護城河降低長期成本並提升競爭護城。但技術壁壘建成後，公司的競爭力和估值提升，可能引來收購方。

| 屬性 | 值 |
|------|---|
| **可用年份** | Y2+ |
| **啟動門檻** | 組織心跳 > 55 |
| **維護門檻** | 組織心跳 > 50（里程碑時檢查） |
| **預付成本** | $100 萬 |
| **維護費** | +$15 萬/季 |
| **工期** | 4 季 |

**完成獎勵（未加碼）：**
- 固定支出永久 **-$10 萬/季**
- 組織心跳 **+5**

**完成獎勵（加碼）：**
- 固定支出永久 **-$15 萬/季**
- 組織心跳 **+8**
- 本局剩餘時間：**1 張熵牌自動以半價化解**（系統自動選最重的那張）

**伏筆種子（完成時埋入，+4 季觸發）：**
```
觸發延遲：+4 季
落點類型：特殊事件（非普通數值調整）
效果：
  「技術壁壘吸引了外部收購方。某私募基金提出以現金 $300 萬收購公司。」

  → 玩家選擇「接受」：
    遊戲立即結束（若在 Y5 前，視為提前退出）
    最終估值 = 當前估值分數 + 將 $300 萬折算入現金資產分
    顯示「戰略退出」結局頁

  → 玩家選擇「拒絕」：
    客戶根系 +5，組織心跳 +3
    （「公司宣佈拒絕收購，團隊凝聚，客戶信心增強」）

歸因標籤：技術壁壘引發收購邀約
溢出處理：若落點超出 Y5Q4，在末季以伏筆形式呈現「你的技術已吸引外部目光」（僅敘事，不觸發收購事件）
```

**現金流分析：**
```
預付 $100 + 維護 4×$15 = $160 萬總投入
完成收益：-$10 萬/季支出，心跳+5
支出節省：如 Y3 完成，剩 8 季 → 節省 $80 萬
淨成本：$160 - $80 = $80 萬（換來心跳+5 + 熵牌化解）
```

### P3：人才梯隊（Talent Pipeline）

**核心商業邏輯：** 系統性培養內部人才，防止關鍵人才流失帶來的組織崩潰。這是防守型投資——不讓你飛得更高，但防止你跌太深。

| 屬性 | 值 |
|------|---|
| **可用年份** | Y3+ |
| **啟動門檻** | 現金動脈 > 50% |
| **維護門檻** | 現金動脈 > 40%（里程碑時檢查） |
| **預付成本** | $90 萬 |
| **維護費** | +$15 萬/季 |
| **工期** | 4 季 |

**完成獎勵（未加碼）：**
- 組織心跳永久**地板 = 30**（本局剩餘時間，心跳不會低於 30）
- 組織心跳 **+8**

**完成獎勵（加碼）：**
- 組織心跳永久**地板 = 35**
- 組織心跳 **+12**
- 心跳自然衰減 **-2 → -1/季**，持續 4 季

**伏筆種子：無**

> 設計說明：人才梯隊是唯一無伏筆的計畫。這是刻意的——它是「保險單」，不是「成長引擎」。獎勵是防守型（地板），沒有進攻型（收入/根系）。玩家在危機感高漲時最容易被吸引，但此計畫需要現金動脈充足（>50%）才能啟動，這本身就是一個反直覺的設計：最需要保險的時候，往往已經買不起了。

**現金流分析：**
```
預付 $90 + 維護 4×$15 = $150 萬總投入
完成收益：心跳地板 30 + 心跳+8
無直接現金回報——純粹的「防跌」設計
```

**與 AI 對手互動（模組 B）：**
狼群型 AI（砸錢挖人才）效果：若活躍此計畫，心跳被挖角衝擊減半。

### P4：品牌升級（Brand Upgrade）

**核心商業邏輯：** 品牌重塑提升市場定位，但同時拉高了客戶期望值。若服務能力未同步提升，品牌升級反而加速客戶流失。

| 屬性 | 值 |
|------|---|
| **可用年份** | Y3+ |
| **啟動門檻** | 客戶根系 > 55 |
| **維護門檻** | 客戶根系 > 50（里程碑時檢查） |
| **預付成本** | $90 萬 |
| **維護費** | +$15 萬/季 |
| **工期** | 4 季 |

**完成獎勵（未加碼）：**
- 客戶根系 **+8**
- 季收入永久 **+$15 萬**
- 飛輪③觸發門檻從「根系 > 55」降為「根系 > 50」

**完成獎勵（加碼）：**
- 客戶根系 **+12**
- 季收入永久 **+$20 萬**
- 飛輪③觸發門檻降為「根系 > 45」

**伏筆種子（完成時埋入，+2 季觸發）：**
```
觸發延遲：+2 季
落點類型：持續效果窗口（4 季）
效果：
  「品牌升級已向市場傳遞更高期望值。客戶評審標準提高。」
  根系自然衰減：-1/季 → -2/季，持續 4 季

  條件檢查（每季衰減期間）：
  → 若根系維持 ≥ 65：衰減恢復 -1/季，期末根系 +5（「品牌承諾兌現，口碑升級」）
  → 若根系未維持 ≥ 65：-2/季 持滿 4 季（「品牌承諾落空，客戶流失加速」）

歸因標籤：品牌升級後的期望管理
```

**現金流分析：**
```
預付 $90 + 維護 4×$15 = $150 萬總投入
完成收益：根系+8，季收入+$15萬，飛輪門檻降低
伏筆風險：若根系無法維持 ≥65，等於完成後 4 季多衰減 -4 總計
淨效益取決於玩家能否在完成後維護根系
```

### P5：供應鏈優化（Supply Chain Optimization）

**核心商業邏輯：** 精實供應鏈降低成本，但韌性下降。在正常環境下效益顯著，但一旦遭遇供應中斷，損失加倍。

| 屬性 | 值 |
|------|---|
| **可用年份** | Y2+ |
| **啟動門檻** | 現金動脈 > 45% |
| **維護門檻** | 現金動脈 > 35%（里程碑時檢查） |
| **預付成本** | $70 萬（五個計畫中成本最低） |
| **維護費** | +$15 萬/季 |
| **工期** | 4 季 |

**完成獎勵（未加碼）：**
- 固定支出永久 **-$15 萬/季**
- 一次性現金 **+$40 萬**（庫存優化釋放流動資金）

**完成獎勵（加碼）：**
- 固定支出永久 **-$20 萬/季**
- 一次性現金 **+$60 萬**
- 本局剩餘時間：現金類熵牌「硬扛」成本 **-30%**

**伏筆種子（完成時埋入，+3 季窗口）：**
```
觸發條件：完成後 3 季內，若觸發熵牌 E-04（關鍵設備故障）或 E-02（原物料價格暴漲）
效果：觸發的熵牌效果翻倍（精實供應鏈增大脆弱性）
若 3 季內未觸發 E-04/E-02：無後果

歸因標籤：供應鏈精實帶來的脆弱性
```

**現金流分析：**
```
預付 $70 + 維護 4×$15 = $130 萬總投入
完成回收 $40 萬 → 淨現金成本 $90 萬
固定支出 -$15萬/季 → 6 季回本
若 Y2Q1 啟動（Y2Q4 完成），Y4Q2 回本
若 Y4Q1 啟動（Y4Q4 完成），Y5Q2 回本（只賺 2 季節省）

最佳啟動時機：越早越好，收益期越長
```

**熵牌交互：**
E-08（現金流斷鏈）活躍期間：活躍計畫的維護費當季翻倍（+$30 萬）。

## 與現有系統整合規則

### 八步結算更新

| 結算步驟 | 新增邏輯 |
|---------|---------|
| **Step 1：投資效果** | 計畫啟動 → 扣預付金；計畫加碼 → 扣加碼費；計畫救援 → 扣救援費 $30 萬 |
| **Step 5：正常營運** | 維護費加入固定支出（計畫活躍期間每季 +$15 萬）；里程碑檢查（Q2/Q4）；完成時交付獎勵 |
| **Step 3：伏筆檢查** | 計畫伏筆種子使用現有伏筆佇列（格式與決策牌伏筆相同） |

### 命脈門檻鎖定更新（§2.10 擴充）

| 狀態 | 計畫限制 |
|------|---------|
| 所有命脈 🟢 | 可啟動任何符合年份和門檻的計畫 |
| 任一命脈 🟠/🔴 | **不能啟動新計畫**（危機時不應開新戰線） |
| 進行中計畫 | 不受命脈鎖定影響（計畫自動進行；只有里程碑時的門檻檢查影響是否停滯） |
| 救援動作 | 受現有現金投資鎖定規則約束（動脈 <30% 無法救援） |

### 年度結算預告（§5.5 擴充）

## 模擬器整合規格

### 風格對計畫的行為映射

| 玩家風格 | 計畫行為 |
|---------|---------|
| 保守型 | 不啟動計畫（現金保留優先） |
| 平衡型 | 動脈 >60% 時，第一個可用的計畫季度啟動 |
| 激進型 | 動脈 >45% 時，嘗試啟動最便宜的計畫 |
| 聰明激進型 | 動脈 >55% 時，選對應最弱命脈的計畫啟動 |

### 驗證目標

跑 8 局（4 風格 × 2 牌組輪替）後確認：
- [ ] 存活率維持 6/8（計畫不應改變死亡率）
- [ ] 平衡型/聰明激進型至少各完成 1 個計畫
- [ ] 計畫伏筆在 ≥3 局中觸發至少 1 次
- [ ] 無計畫單獨導致動脈歸零
