# CEO Review：畫面豐富度 + 互動卡牌內容規劃

> **日期：** 2026-04-04
> **角色：** CEO / Product Owner 視角
> **目標：** 讓遊戲從「能玩的原型」升級到「想分享的產品」

---

## 現況差距分析

### 畫面豐富度：目前 6/10

| 問題 | 嚴重度 | 原因 |
|------|:------:|------|
| 所有 emoji 當 icon（🐯🔶💰） | 高 | 跨平台渲染不一致，顯得業餘 |
| 卡片類型視覺無差異 | 高 | 決策/熵/執行/商機長一樣，只靠小 tag 區分 |
| 數字變化沒有動畫 | 中 | +30 或 -80 出現就是靜態文字，沒有衝擊感 |
| 沒有季度轉場 | 中 | 季與季之間無過渡，感覺像在填表 |
| Dashboard 圖表太基礎 | 低 | 純 SVG path，沒有互動或 tooltip |

### 互動卡牌：目前 7/10

| 問題 | 嚴重度 | 缺什麼 |
|------|:------:|--------|
| 沒有「事件鏈」 | 高 | 決策之間缺乏因果串連感，每張牌像獨立事件 |
| 缺少「時機型」決策 | 中 | 所有決策都是「選 A 還是 B」，沒有「現在做還是等」 |
| 沒有「議價 / 談判」機制 | 中 | 現實中 CEO 很多時間在談判，遊戲裡沒有 |
| 缺少「人物角色」 | 中 | 卡片裡的「業務主管」「財務主管」沒有身份，沒有累積 |

---

## 方案規劃（三個層級）

### 🥇 P0：視覺體驗跳級（影響最大，用戶第一眼感知）

#### 1. 卡片類型視覺分層

每種卡片有獨特的視覺語言，一眼辨識：

| 卡片類型 | 邊框顏色 | 頂部漸層 | 圖標風格 |
|---------|---------|---------|---------|
| 決策牌 | cyan 漸層邊框 | 深藍→透明 | ⚖️ 天秤 SVG |
| 熵牌 | 紅色脈搏邊框 | 暗紅→透明 | 🌀 旋渦 SVG |
| 執行卡 | 綠色邊框 | 深綠→透明 | ⚡ 閃電 SVG |
| 商機牌 | 金色邊框 | 金→透明 | ✨ 星 SVG |
| AI 事件 | 紫色邊框 | 紫→透明 | 🤖 齒輪 SVG |
| 意外支出 | 紅色實線邊框 | 紅色警告條 | ⚠️ 三角 SVG |

**實作方式：** CSS 漸層 border + 頂部 8px 色帶 + inline SVG icon（不用外部字體）

#### 2. 數字滾動動畫

命脈數字變化時用 CSS counter + transition，從舊值滾動到新值：

```css
.ll-val { transition: all .4s cubic-bezier(.4,0,.2,1); }
```

結算畫面的 delta 數字用 typewriter 效果逐項出現（每項 150ms 延遲）。

#### 3. 季度轉場

季與季之間加入 0.5 秒的淡入淡出：
- 舊內容 fadeOut 0.2s
- 短暫黑屏 0.1s
- 新內容 fadeIn 0.3s
- 側邊欄數字同步更新

#### 4. 選項按鈕差異化

| 選項 | 視覺提示 |
|------|---------|
| 花錢型（cash < 0） | 左邊紅色 2px 邊線 |
| 賺錢型（cash > 0） | 左邊綠色 2px 邊線 |
| 高風險（有伏筆） | 右上角紫色 ⏳ 標記 |
| 被鎖定 | 紅色鎖頭圖標 + 半透明灰 |

### 🥈 P1：互動內容深化（增加決策深度和重玩性）

#### 5. 「角色系統」— 給 NPC 名字和個性

在決策牌中反覆出現同一個人物，建立情感連結：

| 角色 | 出現在 | 個性 | 設計意圖 |
|------|--------|------|---------|
| 林主管（業務） | #002, #003, #010, #014, #018 | 衝勁強但容易衝動 | 代表「進攻派」觀點 |
| 陳主管（財務） | #004, #007, #008, #017 | 保守謹慎 | 代表「防守派」觀點 |
| 張總監（技術） | #015, #016, #011 | 沉默但關鍵 | 代表「長期主義」觀點 |
| 王顧問（外部） | AI 事件、挑戰牌 | 客觀冷靜 | 提供第三方視角 |

**實作方式：** 卡片描述中出現角色名時高亮顯示。覆盤時統計「你最常聽誰的建議」。

#### 6. 「時機型」決策 — 新增 5 張

不是「A 或 B」而是「現在做還是等下季做」：

| 牌名 | 場景 | 現在做 | 等一季 |
|------|------|--------|--------|
| 設備採購窗口 | 供應商打折但只限本季 | 便宜 30% 但佔用現金 | 原價但保留靈活度 |
| 人才市場低谷 | 大公司裁員，市場上好人才多 | 低成本搶人 | 可能被搶走 |
| 政策補助截止 | 補助申請截止下季 | 花時間申請 | 錯過就沒了 |
| 競品弱勢期 | 競品出問題，客戶在觀望 | 趁機搶客 | 客戶自己會回來嗎？ |
| 匯率好轉 | 原物料進口成本暫時降低 | 囤貨省成本 | 佔用倉儲和現金 |

**設計重點：** 選「等」不是免費的——下季該牌消失，機會永遠錯過。

#### 7. 「議價 / 談判」機制 — 新增 3 張

玩家選擇談判策略，影響最終結果：

| 牌名 | 談判對象 | 策略選項 |
|------|---------|---------|
| 大客戶續約談判 | 最大客戶 | 強硬（風險高回報高）/ 讓步（穩但利潤低）/ 創意方案（第三條路） |
| 供應商議價 | 核心供應商 | 威脅換人 / 長約鎖價 / 合資降本 |
| 投資人條件 | 外部投資人 | 接受估值折讓 / 堅持底線 / 可轉債方案 |

#### 8. 「連鎖事件」— 特定決策觸發後續劇情

目前級聯牌只有 #003 和 #013。擴充到更多卡片有後續：

| 觸發決策 | 後續事件（2-3 季後） |
|---------|-------------------|
| 裁員（#004 選 A） | 「前員工在社群控訴裁員」事件 |
| 全面透明（#008 選 A） | 「媒體專訪：危機處理典範」正面事件 |
| 內部創業成功（#009 選 A 成功） | 「子公司要獨立，你放手嗎？」新決策 |

### 🥉 P2：沉浸感升級（讓玩家忘記自己在「玩遊戲」）

#### 9. 「季報」敘事 — 每 4 季一段故事

Q4 年度結算時，不只顯示數字，而是生成一段 3-4 句的經營回顧：

> 「${公司名} 的第二年像是坐雲霄飛車。年初的股權風波差點撕裂團隊，但你選擇了動態股權方案，三季後成為全公司最認同的制度。客戶根系從 55 攀升到 82——這是老客戶口碑帶來的。但財務主管的警告還在耳邊：現金只夠撐兩季。第三年，你準備好了嗎？」

**實作方式：** 根據本年度的關鍵決策 + 命脈變化動態生成。

#### 10. 「CEO 日記」— 可選的沉浸式模式

每做一個決策後，顯示一句 CEO 第一人稱日記：

- 選了裁員：*「簽下裁員通知的那一刻，辦公室安靜得像停電了。」*
- 選了逆勢加碼：*「所有人都說我瘋了。但我看到的是：危機結束後，留下來的人值得十倍。」*
- 買了第一個資產：*「被動收入 $8 萬/季。不多，但每一塊錢都不用我操心。」*

**實作方式：** 每張卡每個選項附一句 diary 字串，決策後淡入顯示。

## 優先順序推薦

| 優先 | 項目 | 預估工時 | 影響 |
|:----:|------|:-------:|:----:|
| 🟢 1 | 卡片視覺分層（#1） | 1h | 第一眼質感提升 50% |
| 🟢 2 | 選項按鈕差異化（#4） | 30min | 決策可讀性大幅提升 |
| 🟢 3 | 時機型決策 5 張（#6） | 1h | 決策維度從 2 種變 3 種 |
| 🟡 4 | 議價/談判牌 3 張（#7） | 45min | 新的互動模式 |
| 🟡 5 | 季度轉場動畫（#3） | 30min | 節奏感提升 |
| 🟡 6 | 角色系統（#5） | 1h | 情感連結 |
| 🟡 7 | 連鎖事件 3 組（#8） | 45min | 因果感提升 |
| 🔵 8 | 季報敘事（#9） | 1h | 沉浸感飛躍 |
| 🔵 9 | CEO 日記（#10） | 1.5h | 角色代入感 |
| 🔵 10 | 數字動畫（#2） | 30min | 微體驗優化 |

**建議先做 🟢 四項（卡片分層 + 按鈕差異化 + 時機牌 + 談判牌），這四項加起來約 3 小時，但體感提升最大。**
