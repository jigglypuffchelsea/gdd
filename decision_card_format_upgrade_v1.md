### #002 明星員工求去 → 三路

原 A/B 不變，新增選項 C。

**選項 C：不挽留，啟動接班培養計畫**

```
├ 現金 -$50 萬（招募+培訓接班人）
├ 組織心跳 -6（明星離開，短期震盪）
├ 客戶根系 -4（該員工的客戶關係暫時斷層）
├ ⏳ 伏筆已埋入時間軸
├ 執行門檻：無
└ 鎖定提示：N/A
```

**選 C 伏筆：**

```
觸發延遲：+4 季
落點類型：固定
數值效果：
  ├ 組織心跳 +10（接班人成熟，證明「公司不靠一個人」，團隊安全感提升）
  └ 客戶根系 +5（接班人帶來新風格，部分客戶反而更滿意）
歸因標籤：組織心跳 ▲ 大「接班梯隊成熟，組織韌性提升」/ 客戶根系 ▲ 小「新客戶經理帶來新活力」
溢出處理：×0.7 末季清算
```

**三路對照：**

| | A 加薪留人 | B 虛擬股權 | C 放手培養 |
|---|-----------|-----------|-----------|
| 即時現金 | -$80 萬 | -$30 萬 | -$50 萬 |
| 即時心跳 | +3 | +8 | -6 |
| 即時根系 | 不變 | -5 | -4 |
| 長期心跳 | -12（公平危機） | +12（穩定） | +10（韌性） |
| 核心邏輯 | 花大錢保人 | 花小錢改制度 | 花中錢建梯隊 |
| 窮人版？ | 否——最貴 | 否——最聰明 | 是——接受損失換長期結構 |

**設計意圖：** C 是「承認個人英雄主義的局限」。短期最痛（心跳和根系都跌），但長期建立了不依賴單一個人的組織。對應知識點：梯隊建設。

### #006 股權激勵要求 → 三路

原 A/B 不變，新增選項 C。

**選項 C：口頭承諾「明年再談」，先穩住人**

```
├ 現金 $0
├ 組織心跳 +2（元老暫時被安撫——「老闆說了明年會處理」）
├ ⏳ 伏筆已埋入時間軸
├ 執行門檻：無
└ 鎖定提示：N/A
```

**選 C 伏筆：**

```
觸發延遲：+3 季
落點類型：固定
數值效果：
  ├ 組織心跳 -15（元老等了三季什麼都沒發生，信任徹底崩塌：「老闆畫大餅」）
  └ 客戶根系 -5（核心元老開始消極怠工，客戶服務品質下滑）
連帶觸發：1 位元老離職事件（組織心跳再 -5）
歸因標籤：組織心跳 ▼ 大「承諾未兌現，核心團隊信任崩塌」
溢出處理：×0.7 末季清算
```

**三路對照：**

| | A 給實股 | B 動態虛擬股權 | C 口頭承諾拖延 |
|---|---------|-------------|------------|
| 即時現金 | $0 | -$40 萬 | $0 |
| 即時心跳 | +10 | +5 | +2 |
| 長期心跳 | -8（股東心態） | +8（績效綁定） | -15（信任崩塌） |
| 核心邏輯 | 大方但埋雷 | 花錢建制度 | 零成本但代價最高 |

**設計意圖：** C 是最常見的老闆真實反應——「先拖著」。零成本的誘惑，但代價是團隊信任的不可逆損傷。教學點：拖延不是決策，是最差的決策。

### #008 跨界競爭者進入 → 三路

原 A/B 不變，新增選項 C。

**選項 C：主動接觸競爭者，提出策略聯盟（成為其外包夥伴）**

```
├ 現金 -$40 萬（商務談判 + 法律費用）
├ 客戶根系 -3（部分客戶聽到你跟對手合作，疑慮：「你到底站哪邊」）
├ 組織心跳 +2（團隊覺得「化敵為友」很聰明）
├ ⏳ 伏筆已埋入時間軸（隨機）
├ 執行門檻：客戶根系 > 45
└ 鎖定提示：若不符合 →「客戶基礎太弱，沒有談判籌碼」
```

**選 C 伏筆：**

```
觸發延遲：+4 季
落點類型：隨機
├ 聯盟成功（50%）：季收入 +$40 萬（穩定外包訂單），客戶根系 +8（借對手品牌擴大市場）
└ 聯盟破裂（50%）：競爭者學完你的技術後自己做，客戶根系 -10，季收入 -$30 萬
歸因標籤（成功）：客戶根系 ▲ 中「策略聯盟帶來新客源」
歸因標籤（失敗）：客戶根系 ▼ 大「聯盟破裂，技術外洩」
溢出處理：×0.7 末季清算
```

**三路對照：**

| | A 降價迎戰 | B 深耕壁壘 | C 策略聯盟 |
|---|-----------|-----------|-----------|
| 即時現金 | 利潤率壓縮 | -$100 萬 | -$40 萬 |
| 即時根系 | +3 | -6 | -3 |
| 長期根系 | -12（價格陷阱） | +15（忠誠壁壘） | +8 或 -10（賭博） |
| 核心邏輯 | 跟價搏命 | 走自己的路 | 把敵人變盟友 |

**設計意圖：** C 是成本最低的選項但風險極高——跟虎謀皮。引入「合作-背叛」賽局論維度。教學點：聯盟的前提是你有不可替代的價值，否則你只是在教對手。

### #014 海外大單 → 三路

原 A/B 不變，新增選項 C。

**選項 C：接單但不設點——遠端服務 + 派駐短期團隊**

```
├ 現金 -$60 萬（短期派駐 + 遠端基礎設施）
├ 應收帳款 +$200 萬（到帳需 2 季，同選項 A）
├ 組織心跳 -2（比 A 的 -5 輕——不用長期外派）
├ 客戶根系 +1（有國際客戶但不如設點有說服力）
├ ⏳ 伏筆已埋入時間軸
├ 執行門檻：現金動脈 > 45%
└ 鎖定提示：若不符合 →「現金不足」
```

**選 C 伏筆：**

```
觸發延遲：+4 季
落點類型：固定
數值效果：
  ├ 應收 $200 萬中實際到帳 $140 萬（遠端服務品質問題導致扣款更多）
  ├ 客戶不續約——無持續收入增加
  └ 但組織心跳 +3（團隊學到國際經驗，無長期外派負擔）
歸因標籤：現金動脈 ▲ 小「海外訂單部分到帳」/ 組織心跳 ▲ 小「國際經驗積累」
溢出處理：×0.7 末季清算
```

**三路對照：**

| | A 設點全力做 | B 婉拒專注國內 | C 遠端輕模式 |
|---|------------|-------------|------------|
| 即時現金 | -$150 萬 | $0 | -$60 萬 |
| 即時心跳 | -5 | +3 | -2 |
| 長期收入 | +$50 萬/季（持續）| +$20 萬/季 | 無持續收入 |
| 長期成本 | +$30 萬/季（海外） | 無 | 無 |
| 核心邏輯 | 重注押國際化 | 聚焦本業 | 嚐一口但不嚼 |

**設計意圖：** C 看起來是聰明的「低風險試探」，但實際上兩頭不到岸——投入不夠拿不到長期價值，又比完全不做花了一筆錢。教學點：半吊子的國際化比不做更浪費。

### #017 數據安全事件 → 三路

原 A/B 不變，新增選項 C。

**選項 C：符合最低法律義務——通報主管機關 + 個別通知受影響客戶，不公開聲明也不全額補償**

```
├ 現金 -$80 萬（資安修補 + 法律合規 + 個別通知成本）
├ 客戶根系 -3（受影響客戶知道了但處理態度不算積極）
├ 組織心跳 +1（至少沒有隱瞞——比 B 好一點）
├ ⏳ 伏筆已埋入時間軸
├ 執行門檻：無
└ 鎖定提示：N/A
```

**選 C 伏筆：**

```
觸發延遲：+3 季
落點類型：固定
數值效果：

  └ 無額外罰款（畢竟合規了）
歸因標籤：客戶根系 ▼ 中「最低限度處理，客戶信任緩慢流失」
溢出處理：×0.7 末季清算
```

| | A 全面透明 | B 低調處理 | C 最低合規 |

| 核心邏輯 | 短痛換長贏 | 省錢埋炸彈 | 不犯法但也不加分 |

**設計意圖：** C 是最「正常」的企業反應——做到合規義務、控制成本。長期後果不如 B 慘烈，但也得不到 A 的信任反彈。教學點：合規是底線，不是策略；危機處理的差異在於你做的比義務多多少。

## 級聯型（2 張）

級聯規則：選了 A 或 B 後，**下一個決策季的首位牌**固定為該選項的子決策牌。不打亂季度節奏——只是替換原本排在首位的牌（被替換的牌回到牌池底部）。

### #003 年度目標拉鋸 → 級聯

原牌不變。選 A 或 B 後，下季首位牌替換為對應子牌。

#### #003-A1 子牌：目標降了，降多少？

**觸發條件：** 上季 #003 選了 A（降低目標）

> 你決定降目標了。業務主管問：「降 20% 還是降 10%？降 10% 其實也沒什麼意義，但降 20% 怕團隊習慣了。」

**選項 A1：降 20%（原方案）**

```
├ 效果：維持 #003 選 A 的原始伏筆不變
├ 即時：無額外效果
└ 設計意圖：確認原決策，子牌只是讓你「再想一次」
```

**選項 A2：只降 10%（折衷）**

```
├ 組織心跳 -2（團隊覺得「不上不下」，降了又不夠）
├ 客戶根系 +1（目標降幅小，業務動力只微降）
├ 修改原伏筆：觸發延遲 +4 季的懲罰減半
│   心跳 -4（原 -8）、根系 -3（原 -6）
└ 設計意圖：折衷看起來聰明但效果最差——兩邊都沒滿足
```

#### #003-B1 子牌：增量算法怎麼設計？

**觸發條件：** 上季 #003 選了 B（維持目標 + 增量績效）

> 獎金池準備好了。HR 問你：超額獎金怎麼算？

**選項 B1：線性提成（超額部分按固定比例發獎金）**

```
├ 現金影響：可預期——每季超額獎金約 $20-30 萬
├ 組織心跳 +3（規則簡單，團隊好理解）
├ 修改原伏筆效果（+4 季）：心跳 +8（原 +10，簡單但上限低）、根系 +3（原 +5）
└ 設計意圖：簡單好執行，但激勵效果有天花板
```

**選項 B2：階梯加速（超額越多，提成比例越高）**

```
├ 現金影響：上限不確定——頂尖業務可能拿到極高獎金
├ 組織心跳 -1（規則複雜，部分人覺得不公平：「只有大業務有機會」）
├ 修改原伏筆效果（+4 季）：心跳 +12（原 +10，激勵效果更強）、根系 +8（原 +5），但現金 -$60 萬（原 -$40 萬）
└ 設計意圖：高風險高回報——頂尖人才瘋狂衝刺，但團隊可能分化
```

### #013 合夥人攤牌 → 級聯

原牌不變。選 A 或 B 後，下季首位牌替換為對應子牌。

#### #013-A1 子牌：線上業務怎麼做？

**觸發條件：** 上季 #013 選了 A（同意投 $300 萬衝線上）

> 合夥人重新振作了。但 $300 萬怎麼花？他提了兩個方案：

**選項 A1：自建平台（全部自己開發）**

```
├ 即時：無額外現金支出（$300 萬已包含）
├ 組織心跳 +3（技術團隊興奮：「終於做自己的產品」）
├ 修改原伏筆（+5 季隨機結果）：
│   成功機率提升至 70%（自建掌控力高），成功時收入 +$100 萬/季
│   失敗時虧損更大：現金 -$150 萬（開發成本沉沒更多）
└ 設計意圖：高控制力但失敗代價巨大
```

**選項 A2：找現有平台合作（用第三方基礎設施）**

```
├ 即時：現金回收 +$80 萬（$300 萬預算只用 $220 萬）
├ 組織心跳 -2（合夥人稍有不滿：「這不是我想做的方式」）
├ 修改原伏筆（+5 季隨機結果）：
│   成功機率降至 50%（依賴第三方有風險），成功時收入 +$60 萬/季（利潤被分潤）
│   失敗時虧損較小：現金 -$50 萬
└ 設計意圖：降低風險也降低回報，但合夥人不完全認同
```
