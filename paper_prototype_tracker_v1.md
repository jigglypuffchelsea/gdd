## 三局設定

每局用不同的玩家風格，覆蓋不同計畫：

| 局 | 玩家風格 | 目標計畫 | 啟動時機 | 設計意圖 |
|----|---------|---------|---------|---------|
| 1 | 平衡型 | 供應鏈優化（P5） | Y2Q1（最早可能） | 最便宜的計畫 + 最早啟動 = 驗證最佳情境 |
| 2 | 保守型 | 人才梯隊（P3） | Y3Q1 | 防守型計畫 + 保守風格 = 驗證現金壓力容忍度 |
| 3 | 激進型 | 新市場開拓（P1） | Y2Q3 | 激進風格 + 有伏筆的計畫 = 驗證最危險組合 |

### 前置條件

```
起始值：同上
計畫：P3 人才梯隊

  維護門檻：動脈 > 40%

  維護費：$15 萬/季
  完成：心跳地板 = 30 + 心跳 +8
  伏筆：無
```

### 保守型行為規則

```
投資：每季只投 1 項，優先投最便宜的

加碼：不加碼
放棄：動脈跌破 50% 就主動放棄（保守心態）
```

### 設計問題清單

跑完後逐條檢查：

```
□ 有沒有任何一局因「計畫維護費」直接壓死動脈？
  → 如果有：維護費 $15 萬太高，降到 $10 萬
  
□ 里程碑停滯是否都合理？（命脈確實低於門檻才停滯）
  → 如果門檻太容易跌破：降低維護門檻 5 點
  → 如果門檻永遠達標：提高維護門檻 5 點
  
□ 伏筆種子是否有機會觸發？
  → 如果 3 局都沒觸發：延遲季數減少 1 季，或觸發條件放寬
  → P1 伏筆條件「心跳 < 55」太嚴或太鬆？
  
□ 計畫完成獎勵是否改變了遊戲走向？
  → 太強：降低數值（根系/收入）
  → 太弱：玩家不會選計畫，提高獎勵
  
□ 放棄機制是否合理？
  → 心跳 -5 太重：保守型可能永遠不敢啟動
  → 冷卻 2 季太長：激進型可能衝第二個計畫的機會被卡死
  
□ 加碼時機（Q2 里程碑）是否有決策張力？
  → 花 $50 萬 + 1 格的代價 vs 強化獎勵 — 值得猶豫嗎？
```

## 快速決策牌/熵牌抽牌表（簡化版）
