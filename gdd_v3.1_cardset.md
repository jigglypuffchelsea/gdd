### 牌 #002：明星員工求去

#### ① 情境描述

**牌型：** 事件型（變體 A）
**格式：** 三路選擇
**觸發時機：** Y2–Y4 任意決策季，組織心跳 40–65

> 你的業務總監站在門口，臉色不太好。
> 「老闆，我說實話，外面有人開 1.5 倍薪水找我。我不想走，但家裡壓力很大。你看……能不能想想辦法？」
> 他一個人扛著公司 25% 的業績。

---

#### ② 當下數字

```
可動用現金      $350 萬
應收帳款        $0
本季預計淨額    +$30 萬
──────────────────────
現金動脈        63% 🟡
組織心跳        55 🟡
客戶根系        58 🟡
```

---

#### ③ 決策空間

**選項 A：個別加薪 + 留任獎金**

```
├ 現金 -$80 萬（加薪差額 + 留任獎金一次性支出）
├ 組織心跳 +3（此人留下，短期穩住軍心）
├ ⏳ 伏筆已埋入時間軸
├ 執行門檻：現金動脈 > 40%
└ 鎖定提示：若不符合 →「現金不足」
```

**選項 B：提出虛擬股權方案（全員適用）**

```
├ 現金 -$30 萬（方案設計與啟動成本）
├ 組織心跳 +8（全員看到長期承諾，士氣大振）
├ 客戶根系 -5（該員工猶豫期間，部分客戶服務品質下滑）
├ ⏳ 伏筆已埋入時間軸
├ 執行門檻：無
└ 鎖定提示：N/A
```

**選項 C：不挽留，啟動接班培養計畫**

```
├ 現金 -$50 萬（招募+培訓接班人）
├ 組織心跳 -6（明星離開，短期震盪）
├ 客戶根系 -4（該員工的客戶關係暫時斷層）
├ ⏳ 伏筆已埋入時間軸
├ 執行門檻：無
└ 鎖定提示：N/A
```

---

#### ④ 立即後果

**選 A 後季末面板：**

```
可動用現金      $300 萬（$350 + $30 - $80）
現金動脈        54% 🟡

客戶根系        58 🟡
```

**選 B 後季末面板：**

```

```

**選 C 後季末面板：**

```

現金動脈        59% 🟡
組織心跳        49 🟠（55 - 6，接近危機區！）
客戶根系        54 🟡（58 - 4）
```

**歸因標籤：**

```
選 A：現金動脈 ▼ 中「加薪與留任獎金支出」/ 組織心跳 ▲ 小「核心員工留任」
選 B：現金動脈 ▼ 小「股權方案啟動成本」/ 組織心跳 ▲ 中「全員長期激勵啟動」/ 客戶根系 ▼ 小「員工猶豫期服務波動」
選 C：現金動脈 ▼ 小「接班培養投入」/ 組織心跳 ▼ 中「明星員工離開震盪」/ 客戶根系 ▼ 小「客戶關係斷層」
```

### 牌 #012：組織文化稀釋

#### ① 情境描述

**牌型：** 慢性型（變體 C）
**觸發時機：** Y3–Y5 任意決策季，組織心跳 45–65

> HR 主管拿著一份離職面談紀錄來找你。
> 最近走的三個人都是第一年就加入的老員工。離職原因出奇一致：
> 「公司變了。以前大家是一起拼的戰友，現在新人進來只關心薪資和頭銜。
> 開會的時候，已經沒有人敢說真話了。」
> 你環顧辦公室——人數是創業初期的四倍，但你叫得出名字的不到一半。
